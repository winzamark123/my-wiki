# Home Server

Running your own hardware at home is the gateway drug to understanding infrastructure. A home server usually starts as a spare machine running [[Proxmox]] or bare Docker, and grows into a small platform hosting [[Self-Hosted Services]] — media servers, note apps, and automation.

The interesting tension is between convenience and control. Cloud services are effortless; a home server trades that effort for ownership of your data and the freedom to experiment with things like [[Computer Use Agents]] against your own environment.

## Common starting points

Most people begin with an old desktop or a mini PC. The key early decisions are virtualization strategy (covered in [[Proxmox]]) and networking — especially remote access via [[Tailscale]] or a reverse proxy.
