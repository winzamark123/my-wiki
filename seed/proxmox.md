# Proxmox

Proxmox VE is an open-source virtualization platform that combines KVM virtual machines and LXC containers under one web UI. It is the default choice for a [[Home Server]] that needs to run many isolated workloads.

Its killer features for a homelab are snapshots, easy backups, and clustering. Most [[Self-Hosted Services]] end up running inside LXC containers for their low overhead.
